# Vedora AI Backend

Skeleton backend for Vedora AI.

Run locally:

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
# Backend (FastAPI)

Run (PowerShell):

```powershell
cd Backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m uvicorn main:app --reload
```
